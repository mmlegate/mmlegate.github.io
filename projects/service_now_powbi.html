ServiceNow to Power BI Data Analysis

Essentially this project leverages the Python integration within Power BI to perform an API call to ServiceNow. The result is a data visualization board of key insights from the data pulled from ServiceNow.

The data in question pertained to the instance of ServiceNow from my previous job as a Web Tools Consultant for UW Academic Technologies. For this website, I have used dummy data to avoid any concerns.

Tools Result Code

Tools

ServiceNow

ServiceNow is a cloud-based platform that streamlines and automates business processes across IT, HR, and other departments, improving efficiency and service delivery.

  • To generate the query parameters for the API call, I utilized my familiarity with the Boolean operators used for ServiceNow's search function, e.g. AND, OR, NOT, etc.
  • I would replicate the search I needed in ServiceNow, which gave me the query I needed for the API call.
  • I wanted to generate a report for each consultant summarizing their productivity at work, which required me to filter the search via a UW NetID. The problem is, there are several consultants in the office, and I don't want to manually design each report. Thus I needed additional tools for automation.

Python

Python is a versatile programming language, widely used in web development, data science, AI, and automation, supported by a rich ecosystem of libraries and frameworks.

  • Initially, I thought I could manually search each consultant in ServiceNow, export to Excel, and use Power BI from there. Even then, this requires a lot of manual labor that I wasn't too fond of.
  • I realized that an API Call to ServiceNow using Python would be much easier, which was conducted via Power BI.
  • There was no API documentation at my office, so I had to use trial and error to estimate the paramaters I could use to modify the API call. Luckily, ServiceNow generates query strings for each search you do. This allowed me to learn the format needed for the API call.

Power BI

Power BI is a powerful business analytics tool by Microsoft that allows users to visualize and share insights from their data through interactive dashboards and reports.

  • Power BI has a handy Python integration where you ultimately paste your code within the desktop app. You just need to preprocess your data in your Python code, as the raw data from ServiceNow often has incomprehensible labels and values.
  • I manually designed the report pictured below, but I thankfully only had to do it once. I created a UW theme as it looked nice and on brand. I wanted the result to be customized according to my manager's preferences as well.
  • I then proceeded to edit the Power BI query for each consultant's UW NetID, generating about 30 individual reports. The end result is a concise user-friendly report that summarizes a consultant's lifetime at the company.

Result

Here is a screenshot of the report using some dummy data. I'll describe each section of the report as follows:


  • Top graph depicts the lifetime work output of a consultant. In the example below, we can clearly see the peaks and troughs of tickets that they had worked on on a month-by-month basis.
  • Left pie chart describes the subject or LMS that each ticket focused on. For the example, we can see that the consultant mainly worked on tickets pertaining to Canvas.
  • The middle visualization shows the consultant's first and last ticket that they worked on.
  • Right pie chart depicts the type of task that the consultant worked on. In the example, the consultant primarily worked on requests.


Ultimately, I started this project with the goal of acknowledging each consultant's contributions to the office. I wanted consutlants to be reminded of their progress, and gain the ability to track their productivity for future improvement. As my this job required enrollment at the university, my report also served as a memento for graduating seniors, who would have to move on from UW IT.

I also laid the foundation for future consultants to work on this project after I graduate. I see much potential in Power BI's Python integration to generate live dashboards of ServiceNow records, whether it be for tracking knowledge articles, consultant productivity, or ticket lifetimes.
