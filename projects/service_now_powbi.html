Service Now to Power BI Data Analysis

Essentially this project leverages the Python integration within Power BI to perform an API call to Service Now. The result is a data visualization board of key insights from the data pulled from Service Now.

The data in question pertained to the instance of Service Now from my previous job as a Web Tools Consultant for UW Academic Technologies. For this website, I have used dummy data to avoid any UW data concerns.
